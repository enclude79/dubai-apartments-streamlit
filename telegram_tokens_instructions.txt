ИНСТРУКЦИИ ПО НАСТРОЙКЕ TELEGRAM ТОКЕНОВ

1. Создайте файл .env в корневой директории проекта
2. Добавьте в этот файл следующие строки:

TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_CHANNEL_ID=ваш_идентификатор_канала_или_чата

3. Замените "ваш_токен_бота" на токен вашего Telegram бота
   - Чтобы получить токен, напишите @BotFather в Telegram
   - Следуйте инструкциям для создания нового бота
   - Скопируйте полученный токен

4. Замените "ваш_идентификатор_канала_или_чата" на ID вашего канала или чата
   - Для канала: добавьте вашего бота как администратора канала
   - ID канала обычно имеет формат -100xxxxxxxxxx
   - Чтобы узнать ID, можно использовать бота @getidsbot

5. Сохраните файл .env и запустите streamlit_ngrok_server.py
